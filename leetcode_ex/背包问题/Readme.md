## 背包问题简介
1. 0-1背包问题: 选和不选
2. 完全背包问题: 选任意次数
3. 多重背包问题: 每个可选次数有限且限制不同
4. 混合背包问题: 最大价值
5. 二位费用背包问题: 多维最大价值
6. 分组背包问题: 每组只能选一件
7. 背包问题求方案数.
8. 求背包问题的方案.
9. 有依赖的背包问题.
